//Project Description
Simple webapp for a pokedex of original 150 pokemon.

//How to use
If project is functional, will bring up a modal that displays pokemon's name, sprite, and height. Additional functionality can be implemented 
to include pokemon's types, abilities, and other characteristics.

//Project dependencies
Jquery 3.3.1
popper 1.14.6
bootstrap 4.2.1

//API
App calls on pokemon api found at https://pokeapi.co/
